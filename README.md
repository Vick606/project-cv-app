# 📄 CV Generator 🚀

Welcome to the CV Generator – your one-stop solution for creating professional resumes in a snap! ✨

## 🌟 Features

- 📝 Easy-to-use interface
- 👤 General Information section
- 🎓 Education history
- 💼 Work experience
- 🔄 Edit and preview modes
- 🖨️ Print-ready format
